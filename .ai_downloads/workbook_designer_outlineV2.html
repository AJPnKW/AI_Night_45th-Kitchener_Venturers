Pre-populated from the current repo direction: guided start, prompt improvement, reflection, branching, independent build, and close.

1. Welcome and what tonight is for
intro • 8 min • pathway: all
Goal: Set expectations, lower pressure, explain that the goal is learning how output changes when prompts change.
Learner prompt: Write one thing you hope AI could help you do better by the end of tonight.

Facilitator notes: Keep this calm and short. Emphasize that surprising results and weak results are both useful for learning.
2. Live demo: weak prompt vs stronger prompt

Reflection: What changed between the first result and the second result?
Facilitator notes: Use this to anchor the whole night. Show how audience, constraints, budget, and output format matter.
3. Guided exercise: fix a weak prompt
exercise • 12 min • pathway: all
Goal: Give every learner one successful prompt-improvement loop early in the session.
Learner prompt: Pick one card. Improve the prompt. Paste the result. Note what changed.
Reflection: What did you expect the better prompt to return, and what actually happened?

Facilitator notes: This is the bridge from 'cool output' to real thinking.
