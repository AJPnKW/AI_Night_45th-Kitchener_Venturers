Consent flow

Consent details and parent waiver path.

This page explains the approval model for one supervised Venturer AI and ChatGPT workshop. The participant sign-up page handles workshop setup details. The parent waiver page handles the direct approval form.

Independent site notice: this is a volunteer-run consent path for one event at Hope Lutheran Pres Church, 30 Shaftsbury, not an official organizational registration system.

Minimal field set

  • Participant first name
  • Participant age
  • Participant email
  • Parent / guardian acknowledgement
  • Optional parent / guardian email

Approval steps

  1. Review the parent guide and privacy page.
  2. Use the participant sign-up page if device or ChatGPT setup help is still needed.
  3. Complete the parent waiver form to record approval.
  4. Use the contact listed on the form if follow-up is needed.
