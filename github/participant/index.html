Participant path

What to bring, what to expect, and what happens next.

This Venturers session is a beginner-friendly AI and ChatGPT workshop for about 6 participants with 3 scouters in the room. Everyone follows the same shared plan: short intro, guided demo, guided first task, small project work, and wrap-up sharing.

Before the eventDay-of guideAfter the event

Shared session snapshot

  • Audience: Venturers youth group
  • Location: Hope Lutheran Pres Church, 30 Shaftsbury
  • Room setup: Wi-Fi available, participants bring a laptop or Chromebook
  • Support style: structured, low-friction, mixed ability, with quiet support for 2 neurodivergent learners
  • Date and start time: use the same final reminder shared by Scouter Jenn

Shared workshop flow

The same timeline everyone is following

  • 1Short intro and session framing
  • 2Guided demo with one clear example
  • 3Guided first task so everyone gets an early win
  • 4Independent or small-project work with support
  • 5Wrap-up and optional sharing

Before the Session

  • Bring your laptop or Chromebook and its charger.
  • Read the event reminder so you know the date, start time, and location.
  • Do the optional at-home prep if it was sent out.
  • Complete the readiness check if you still need setup help.
  • Bring one simple idea you might want to build or explore.

During the Session

  • Follow the five-step workshop flow one step at a time.
  • Ask for help if you lose track or need a clearer example.
  • Use ChatGPT as a tool, not as something to trust automatically.
  • Keep private personal information out of prompts.

After the Session

  • Keep any useful prompts, notes, or project ideas.
  • Notice what worked well and what still needs checking.
  • Pick one small next step instead of trying to do everything at once.

What to expect

  • Session purpose: get comfortable using AI and ChatGPT in a practical, beginner-friendly way.
  • Terminology: the same words are used across the workshop: intro, demo, first task, project work, wrap-up.
  • Account/setup: do the optional pre-work and use the readiness page if you need help getting access set up.
  • Good habit: verify important answers instead of trusting them automatically.

If you feel stuck

  • Read the task out loud once.
  • Do one small step, not the whole project at once.
  • Ask for an example or starter prompt.
  • Take the quiet support option if you need less pressure.
