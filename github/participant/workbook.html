Participant Workbook

Private-by-default participant workbook for guided learning, note-taking, and reflection.

Participant workbook model

This workbook is meant to feel like your guided learning notebook for the night. Use it to choose an activity, record what you tried, capture what changed, and keep the notes you want to take away with you.

  • Private by default: notes stay in this browser unless you print or download them.
  • Guided learning path: you can choose a level and category without starting from a blank page.
  • Follow-up coaching: the workbook should help you notice gaps and plan the next prompt improvement.
  • Fast-finisher support: if you move quickly, use the extension section instead of waiting.

How to use this workbook

  • Choose your activity level and category.
  • Use the starter prompt if you need one.
  • Write your improved prompt and what changed.
  • Look at the coaching prompt before you move on.
  • If you finish early, use the extension section.
  • Save, download, or print your workbook before you close the page.

How this workbook works

This workbook is local to this browser on GitHub Pages. Your notes stay on this device unless you choose to print them, save them as PDF, or download a JSON copy.

  • Use Save in this browser if you want to keep working later on the same device.
  • Use Download report if you want a portable copy.
  • Use Print or save as PDF if a leader or family wants a paper-style summary.
  • Pre-event surveys and tracked organizer workflows belong in organizer-issued Google Forms, not in this local workbook.

This workbook is private to this browser unless you choose to save, download, or print it.
