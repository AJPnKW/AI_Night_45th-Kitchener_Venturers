What this step is for

This page holds the image prompt task and the extra challenge space for learners who finish quickly. It belongs to the same workbook, so your notes should carry forward with you.

Image prompt reminders

  • Start with subject, mood, colours, style, and layout.
  • Change one thing at a time if the result feels random.
  • Use the extension space if you finish early instead of waiting.
