90-minute facilitation engine.

This page is designed to be used during the workshop. It gives a tightly bundled minute-by-minute structure, facilitator actions, participant actions, transitions, and adaptive responses.

Adaptive quick rules

  • If ahead: give an extension prompt, not a brand-new branch.
  • If behind: reduce the task to one short step and provide an example.
  • If stuck: read the current step out loud, restate the goal, then offer a starter prompt.
  • If overloaded: lower noise, reduce options, and point to the current numbered step only.

Run of show

0–5

Arrival and settle

Facilitator: greet, point to start screen, confirm devices and Wi-Fi. Participants: sit down, open device, connect. Transition: once most screens are ready, say "We are starting with what AI is and the rules for using it safely."

5–10

What AI is / is not

Facilitator: explain what ChatGPT is, what it does well, and what it gets wrong. State the safety rule: no personal or sensitive information in prompts. Participants: listen, ask short questions. Transition: "Now I’ll show one useful example so you can see what success looks like."

10–15

Live demo

Facilitator: run one visible useful prompt with a simple output. Participants: watch and predict what might happen. Transition: "Next, everyone will do the same first prompt so nobody starts alone."

15–20

Common first task

Facilitator: guide everyone through one shared prompt. Participants: enter the same starter task. Success goal: everyone gets a result. Transition: "You have your first success. Now choose a project direction."

20–30

Choose project pathway

Facilitator: offer 2 to 4 simple pathway choices. Participants: choose one. Adaptive note: for hesitant learners, suggest a default pathway instead of open-ended choice.

30–45

Build block 1

Facilitator: circulate and check for blocked learners first. Fast learners: add one feature or stronger prompt. Behind learners: narrow the goal to one small output. Transition: call a short reset at minute 45.

45–60

Build block 2

Facilitator: use recovery prompts for stuck learners, extension prompts for confident learners. Participants: continue building. Adaptive note: keep returning to "What are you making now?" and "What is your next small step?"

60–75

Refine and prepare to share

Facilitator: help learners polish, simplify, or test what they made. Participants: prepare one thing to show or describe. Transition: "Choose one thing you want to share, even if it is still rough."

75–85

Share-back

Facilitator: invite optional sharing, keep it short, and reinforce effort over polish. Participants: show, describe, or pass. Adaptive note: allow opt-out or written sharing for lower-pressure participation.

85–90

Close

Facilitator: restate the learning goals, remind everyone to verify important answers, and point to one next step. Participants: pack up and keep notes or prompts they want to reuse later.

Recovery prompts for stuck learners

  • Tell me what you are trying to make in one sentence.
  • Give me three smaller first steps.
  • Rewrite this in simpler language.
  • Show one example before giving more options.

Extension prompts for fast learners

  • Add a second version and compare the outputs.
  • Improve the prompt to make the result more specific.
  • Turn the output into a checklist, quiz, poster, or script.
  • Ask the model to critique and improve its own answer.
