For parents and guardians

What this session is, how approval works, and where to complete the waiver.

This workshop is a supervised, beginner-friendly AI and ChatGPT session for Venturers. Youth will move through a short intro, guided demo, guided first task, simple personal project work, and wrap-up sharing.

Independent volunteer support site: this page exists to help families understand one event. It is not an official group or Scouts Canada website.

Shared event snapshot

  • Location: Hope Lutheran Pres Church, 30 Shaftsbury
  • Devices: participants bring a laptop or Chromebook
  • Connectivity: Wi-Fi is available
  • Support: mixed ability group with structured support and supervision throughout
  • Date and start time: use the same final reminder shared by Scouter Jenn

What is ChatGPT in plain language?

ChatGPT is a tool that generates text based on patterns in training data and the prompt it is given. It can help with brainstorming, explaining, outlining, and drafting, but it can also be wrong, incomplete, or overly confident.

What youth will practice

  • Writing clearer prompts.
  • Comparing weak and strong outputs.
  • Noticing when an answer needs checking.
  • Using AI as support, not as a replacement for thinking.

Parent / guardian steps

  • Review the participant page, privacy page, and consent page.
  • Help with home setup if a participant still needs device or ChatGPT access support.
  • Use the participant sign-up page if setup help or workshop-prep information still needs to be shared.
  • Complete the parent waiver for approval and follow-up if needed.
