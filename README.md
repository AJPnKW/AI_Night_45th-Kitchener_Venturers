# AI Night

This website is an independent volunteer support resource for one AI Coding Night with 1st Stanley Park Venture Company. It is not an official Scouts Canada or group-owned website.

If you are a participant or parent, start here:

- Repository home: <https://github.com/AJPnKW/AI_Night_1st-Stanley-Park_Venture-Company>
- Participant guide: <https://github.com/AJPnKW/AI_Night_1st-Stanley-Park_Venture-Company/blob/main/AI_Workshop_4_ventures/web/participant/index.html>
- Parent / guardian guide: <https://github.com/AJPnKW/AI_Night_1st-Stanley-Park_Venture-Company/blob/main/AI_Workshop_4_ventures/web/parent/index.html>
- Consent and approval: <https://github.com/AJPnKW/AI_Night_1st-Stanley-Park_Venture-Company/blob/main/AI_Workshop_4_ventures/web/legal/consent.html>
- Privacy notice: <https://github.com/AJPnKW/AI_Night_1st-Stanley-Park_Venture-Company/blob/main/AI_Workshop_4_ventures/web/legal/privacy.html>

What you can expect:

- clear before, day-of, project, and after-event guidance
- simple AI terms and examples
- a calm, low-clutter layout
- printable pages
- a minimal-data consent approach

This site uses standard HTML, CSS, and JavaScript so it can be maintained later without a framework.
