Facilitator plan for the same shared workshop timeline.

This session is one beginner-friendly AI and ChatGPT workshop for about 6 Venturers with 3 scouters, Wi-Fi available, participant laptops or Chromebooks, optional pre-homework, and mixed-ability learner support including 2 neurodivergent learners.

Important: this page is independent volunteer material and does not imply official ownership by 1st Stanley Park or Scouts Canada.

Shared session timeline

  • 1Short intro
  • 2Guided demo
  • 3Guided first task
  • 4Independent or small project work
  • 5Wrap-up and sharing

Before the Session

  • Confirm the date, start time, and Hope Lutheran Pres Church / 30 Shaftsbury location match the same reminder used by leaders and participants.
  • Check Wi-Fi, participant device assumptions, and any demo setup you need.
  • Review optional pre-homework so the opening does not assume too much.
  • Review readiness responses for device status, ChatGPT access status, comfort level, and project interests.
  • Prepare one clear demo, one guided first task, and one simple fallback example.

During the Session

  • Keep the five-step sequence visible and name each transition out loud.
  • Use plain language and one instruction at a time for the first task.
  • Adapt pacing for mixed ability learners and give quiet support where needed.
  • Keep the room focused on practical ChatGPT use, not abstract theory.

After the Session

  • Close with reflection, optional sharing, and one safe next step.
  • Note any follow-up content or support needs for the organizer and leaders.
  • Confirm the same workshop summary is reflected across the participant and leader pages.

Facilitator notes

  • Session purpose: help learners use AI and ChatGPT in a structured, practical way.
  • Delivery style: calm, low-friction, beginner-friendly, and explicit.
  • Important intake data: device readiness, ChatGPT access readiness, comfort level, and what each participant wants to build.
  • Terminology: keep wording consistent: intro, demo, first task, project work, wrap-up.
  • Learner support: normalize different speeds, quiet help, and optional sharing.

Backup prompts

  • Explain a topic to a beginner using five short bullets.
  • Turn these notes into a one-page checklist.
  • Give me three project ideas ranked from easiest to hardest.
  • Rewrite this in simpler language.
