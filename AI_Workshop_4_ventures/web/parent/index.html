Parent and Guardian Information

Warm, plain-language overview for families.

Browser-first by default

This session is a guided introduction to AI tools in the browser. It is not a programming bootcamp, not a hacking session, and not a hidden software-install exercise.

What to bring

  • Device if available
  • Charger or power cable
  • Optional Ethernet cable
  • Browser access is the main requirement

Required / Optional / Not needed by default

CategoryDetails
RequiredBrowser access, a device if available, and a charger or power cable.
OptionalA family-approved free ChatGPT account and an Ethernet cable if available.
Not needed by defaultLocal VM, local Python, terminal tools, complicated installs, or advanced coding setup.

What your child will do

  • Compare weak and stronger prompts in the browser.
  • Practice improving a prompt with a simple template.
  • Try one image prompt.
  • Reflect on what made the result more useful.

What your child will not be required to do

  • Install a VM by default
  • Use local Python by default
  • Use terminal tools by default
  • Turn the session into a coding class

Privacy, accounts, and shared devices

  • A free account may be enough for the main activity, but families should use their own judgment.
  • Browser-first delivery reduces local device changes.
  • The workbook pages save notes locally in the browser unless the learner chooses to print or download them.
  • Optional advanced installs, if ever used later, are separate and removable.
  • If a learner cannot use an account or device, the fallback pathway still allows participation.

Surveys and feedback

Structured pre-event and follow-up input are planned through organizer-issued Google Forms. That collection path is separate from the public website and separate from the local workbook pages.

How to opt out of optional installs

You do not need to approve optional VM or advanced tooling for the core workshop. Staying on the browser-first route is fully supported.

Questions parents may ask

What is this session?

A guided, browser-based introduction to AI and ChatGPT for practical everyday use.

What is it not?

It is not a programming bootcamp, hacking session, or hidden install exercise.

What if our device is shared?

The browser-first path minimizes device changes. Families can decide whether account sign-in is appropriate.

What if we want no installs?

That is fine. The browser-first and fallback routes still support participation.
