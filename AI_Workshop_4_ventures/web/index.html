AI Night: Browser-First Intro to AI

Guided browser-based introduction to AI, prompt practice, and safe practical use for Scouts and Venturers.

Browser-first AI night for Scouts and Venturers

The main session is a calm introduction to AI in the browser. Learners compare weak and strong prompts, try guided prompt practice, test an image prompt, reflect on what changed, and leave with simple takeaways.

Required / Optional / Not needed by default

TypeWhat it means
RequiredA device with a modern browser, if available, plus a charger or power cable.
OptionalAn Ethernet cable and a family-approved free account if desired.
Not needed by defaultLocal VM, local Python, terminal tools, coding setup, or Codespaces.

Platform note

GitHub Pages is the public front end. Workbook notes, reflections, and printable reports run client-side in the browser. Structured surveys and feedback use organizer-issued Google Forms outside this public site.
