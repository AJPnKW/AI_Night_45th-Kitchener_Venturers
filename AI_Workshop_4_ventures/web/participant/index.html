Ask for support early

4

Keep it practical

Choose one small idea and build it step by step.

If you get stuck

  • Read the current step out loud.
  • Ask for one example, not a full new project.
  • Go back to the last thing that worked.
  • Choose the next smallest step.

If you finish early

  • Improve your prompt so the result is more specific.
  • Make a second version and compare them.
  • Turn your result into a checklist, quiz, or poster.
  • Help by sharing what prompt worked for you.
